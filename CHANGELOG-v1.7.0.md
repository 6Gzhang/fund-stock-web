# 更新日志 v1.7.0 (2026-06-26)

## 修复的问题

### 1. 硅基流动 API Key 配置
- 新增 `backend/Data/module_config.json` 配置文件存储 API Keys
- `data/ai.py` 全面重构，使用硅基流动千问 2.5-7B 作为主模型
- 支持 DeepSeek 作为备用模型
- 新增 `_safe_json_parse` 和 `_normalize_ai_result` 处理 AI 返回的畸变 JSON

### 2. 持仓页面报错 (current_price=0)
- `routes/trade.py` 修复持仓 ETF 代码识别（51/15/16/13 开头）
- 自动区分 ETF 和股票，调用正确的行情接口
- 修复 `cost_price` 字段名不匹配问题（改为 `avg_cost`）

### 3. AI 智能分析失败
- `routes/analysis.py` 第46行 KeyError 修复（使用 `.get()` 代替直接访问）
- 千问模型返回字段名兼容（`recommend` → `recommendation`）
- 支持中文字段名（`操作建议`、`买入理由`、`目标价` 等）

### 4. 智能推荐不再保守
- 放宽筛选条件：价格上限 500 元（原来是 200）
- 涨跌幅范围扩大：-5% ~ +9%（原来是 -3% ~ +7%）
- 成交额门槛降低：2000 万（原来是 5000 万）
- 推荐门槛降低：`net_score >= 0`（原来是 `>= 2`）
- ETF 同样放宽筛选
- 目标价计算更激进：基于布林带上轨 + 近期高点
- 差价明显增大：示例 43%、15.4%、13.6%
